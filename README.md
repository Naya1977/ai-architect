# AI Architect – Executive Setup

This repository contains the complete local technical infrastructure for the
AI Systems Architect – Executive track.

## Purpose
A clean, reproducible, and scalable development environment for:
- AI experimentation
- Data processing
- Vector databases
- API automation
- Future production systems

## Tech Stack
- Python 3.11 (virtual environment)
- OpenAI API
- JupyterLab
- NumPy / Pandas
- FAISS + ChromaDB (Vector DB)
- PostgreSQL client (psql)
- MongoDB Compass (client)
- Postman (API testing)
- Git + GitHub
